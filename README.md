This Python tool scans a list of Bitcoin addresses, decodes them from Base58Check format, analyzes their HASH160 fingerprints, and identifies potentially suspicious or weak-looking addresses.

The goal is to detect addresses that might have been:

generated by faulty scripts,

created from simple or low-entropy private keys,

or used in brainwallet experiments.

Decodes each address:

Performs Base58 decoding.

Verifies the checksum (Base58Check).

Extracts the version byte and the HASH160 (the 20-byte public key hash).

Applies heuristic analysis on each HASH160:

Checks for too few distinct bytes (low entropy).

Detects repeated byte patterns (like many 00 or ff).

Flags small byte value ranges (e.g., all bytes between 0x10â€“0x20).

Detects leading zero bytes (e.g., 000000...).

Calculates bit density (too few or too many bits set).

Marks addresses that trigger any heuristic as suspicious.
